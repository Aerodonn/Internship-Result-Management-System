# Internship-Result-Management-System
Databases &amp; Interfaces (COMP1044 UNMC) coursework assignment by Group 39. Web-based Internship Result Management System to improve efficiency, transparency, and accuracy of evaluating students who undergo industrial training. Allows users to login as a admin or assessor.

The database is 
``` internship_management_system_sql ```
please import it using MAMP/phpMyAdmin or whatever database management server software is preferable.

Preferably, start on the 
```login.php``` page, and fill in the credentials. They can be found in the user_login table, but for testing purposes, here are a few logins:
